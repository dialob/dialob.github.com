Introduction

Introduction

Dialob is an open source platform developed to provide easy way to create, test and manage online dialogs that can be used on any online channel.

Dialob consists of following main modules:

  • Dialob Composer that is a tool for subject matter experts, SMEs, to create and test online dialogs. Dialob Composer is available as a cloud service.
  • Dialob Manager that is a backend service that is integrated to given online channel(s) and executes the dialog defined by Dialob Composer

This documentation provides information how a Dialob Manager can be integrated to a given online channel. Go Dialob to see some live examples.

Technical documentation for Dialob services.

  • How to create a new answering session for a dialog and read submitted answers, please see Session REST API
  • How to implement your own answering UI, please see UI Command Protocol
  • How to embed and extend default answering UI, please see Dialob Client Embedding
  • How to make default answering UI look like you need, please see Dialob Client Style Guide
